# YNF custom Odoo addons

Add this directory to your Odoo `addons_path` (alongside `odoo-18-community/addons`):

```
custom-odoo-addons,odoo-18-community/addons
```

## Modules

| Module | Description |
|--------|-------------|
| `ynf_tiktok_shop` | TikTok Shop Partner API connector (OAuth, orders, inventory, returns, webhooks) |

Requires `ynf_inventory` and `ynf_tiktok_live` from `odoo-18-community/addons`.
